==============================
Detector de Plagio para Moodle
==============================

Instalaci—n
===========

Pasos a seguir
#. Instalar el detector de plagio PlagiarismDetector (es una clase que puede ser llamada desde cualquier script php y no solo desde moodle)
	::
	git clone https://github.com/SyrusAkbary/PlagiarismDetector-PHP
#. Instalar el m—dulo en la carpeta de mod de moodle
	::
	cd /carpeta/de/moodle/mod
	git clone https://SyrusAkbary@github.com/SyrusAkbary/PlagiarismDetector_Mod-Moodle.git plagiarismdetector
#. Instalar el bloque en la carpeta de blocks de moodle
	::
	cd /carpeta/de/moodle/blocks
	git clone https://SyrusAkbary@github.com/SyrusAkbary/PlagiarismDetector_Block-Moodle.git plagiarismdetector

#. Entrar en Notificaciones en el panel principal de administraci—n de Moodle para que se instale el m—dulo.

Eso es todo!
